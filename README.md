# Portfolio

A professional portfolio website showcasing skills, projects, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Layout**: Clean and professional design with smooth animations
- **Well-Formatted Sections**: Includes About, Skills, Projects, and Contact sections
- **Easy to Customize**: Simple HTML structure with integrated CSS styling

## How to View

1. Simply open `index.html` in your web browser
2. Or use a local server for development:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Customization

Edit the `index.html` file to:
- Update your name and title in the header
- Modify the About section with your personal description
- Add your actual skills in the Skills section
- Replace project placeholders with your real projects
- Update contact information with your details

## Structure

- **Header**: Your name and professional title
- **Navigation**: Smooth scrolling to different sections
- **About**: Personal introduction
- **Skills**: Technical skills organized in cards
- **Projects**: Portfolio of your work with technology tags
- **Contact**: Ways to reach you (email, GitHub, LinkedIn)
- **Footer**: Copyright information
